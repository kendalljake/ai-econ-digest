# AI Econ Digest Kit

Files are organized so this project stays separate from others.

## Suggested repo/folder structure
Create a new repository or a top-level folder named `ai-econ-digest/` and place these files there.

## Required GitHub Secrets
- SLACK_WEBHOOK_URL
- LLM_API_KEY
- LLM_MODEL

## Optional GitHub Secrets for adding feeds by email
- IMAP_HOST
- IMAP_USER
- IMAP_APP_PASSWORD
